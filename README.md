# Grid Manipulator Simulator

A interactive web app for simulating manipulations on a grid of samples using Cyrillic command sequences (e.g., ЛПВНОБ for Left, Right, Up, Down, etc.). Built with React, Redux, and Material-UI.

## Features
- Command input and optimization (e.g., compress sequences for efficiency).
- Animated sequence execution with adjustable speed.
- History tracking of manipulations.
- Quick actions: Random samples generation and reset.
- Responsive UI with snackbar notifications.

## Installation
1. Clone the repo: `git clone https://github.com/azimjon-95/manipulator-simulator.git`
2. Install dependencies: `yarn install`
3. Run locally: `yarn start`

## Usage
- Enter commands in the TextField (e.g., "ЛПВНОБ").
- Adjust animation speed with the slider.
- Use quick buttons for random setups or reset.

## Tech Stack
- React + TypeScript
- Redux Toolkit
- Material-UI
- Utilities for command parsing and serialization
